Which Filetypes?


1.7 Describe which are the appropriate filetypes for websites


HTML is an appropriate text based file for websites. When the HTML file is viewed in a text editor it can be read. CSS is a plain text file. CSS files determine the look of the website i.e font, colour, size etc this also includes the location of HTML information on the web page.


Image files can be JPEG and PNG amongst others. Both of these formats hold the data that makes up a picture in completely different ways. Mp3 is an example of an appropriate audio file and webM is an example of a video file.
